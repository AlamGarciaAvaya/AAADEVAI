Watson Assistant

Descripción de la tarea

Recibe texto y devuelve el análisis realizado por Watson Assistant

Propiedades de la tarea

  • Texto - Texto que se analizará por la máquina Watson Assistant
  • User Name - Usuario proporcionada por IBM bluemix al crear el servicio de Watson Assistant
  • Password - Password proporcionada por IBM bluemix al crear el servicio de Watson Assistant
  • Work Space Id - Código del espacio de trabaja al que va dirigido el texto

Input schema

  • textAssistant - Texto que se analizará por la máquina Watson Assistant

Output schema

  • intent - Despliega la intención que se ha definido en el espacio de trabajo de Watson Assistant de acuerdo al texto proporcionado

Prerequisitos

Credenciales proporcionadas por IBM bluemix

Related Boundary Events

No aplica

Casos de uso

Definir la intención de un parrafo para determinar la intención del texto que se analizará
