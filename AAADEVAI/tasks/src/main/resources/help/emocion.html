Natural Language Understanding

Descripción de la tarea

Recibe texto y devuelve el análisis realizado por la Watson Natural Language Understanding

Propiedades de la tarea

  • Texto - Texto que se analizará por la máquina Watson Natural Language Understanding
  • User Name - Usuario proporcionada por IBM bluemix al crear el servicio de Natural Language Understanding
  • Password - Password proporcionada por IBM bluemix al crear el servicio de Natural Language Understanding

Input schema

  • textEmociones - Texto que se analizará por la máquina Watson Natural Language Understanding

Output schema

  • emociones - Despliega el porcentaje de acuerdo a las emociones definidas por Watson (Alegría, Tristeza, Asco, Enfado)

Prerequisitos

Credenciales proporcionadas por IBM bluemix

Related Boundary Events

No aplica

Casos de uso

Definir las emociones de un parrafo para determinar la emoción predominante en el texto que se analizará
