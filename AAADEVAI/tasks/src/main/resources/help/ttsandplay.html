Text To Speech And Play Announcement Help

Descripción de la tarea

Recibe texto y genera un archivo de audio almacenado en un servlet dentro de Breeze para reproducor el audio en una llamada.

Propiedades de la tarea

  • Texto To Speech or Media URI - Texto de entrada para crear el audio que se reproducirá en la llamda o URL para obtener el archivo de audio
  • Handle - El número / dirección para quien se reproducirá el anuncio.
  • Play announcement to:
    • Callid - Reproduce el anuncio al callid. Todas las partes conectadas a la llamada escuchan el anuncio.
    • Called - Reproduce el anuncio solo para la parte llamada.
    • Calling - Play the announcement to the calling party only.
  • Use Locale - Si se marca, la asignación / propiedad de la configuración regional se utiliza para reproducir medios. Si no se proporciona, se utiliza el atributo de configuración regional predeterminado de System Manager.
  • Locale - Lenguaje de solicitud admitido por Avaya Aura® Media Server (MS).
  • Interrupt announcement (optional) - Determina si el anuncio de audio se interrumpe cuando se presiona un dígito. Verdadero (predeterminado) interrumpe el anuncio. Falso no lo hace.
  • Duration (secs) - La duración de una solicitud de llamada en segundos.
  • Iterate count - El número de veces que se reproduce el anuncio.
  • Announce forever (Optional) - Configura los medios para que se reproduzcan para siempre sin ser interrumpidos por el vencimiento de un temporizador. Los parámetros de duración y recuento de iteraciones no se utilizan cuando este indicador se establece en verdadero. El valor predeterminado es falso.
  • Cloud Provider- Opción para seleccioner el tipo de proveedor para realizar Text To Speech
    • Google
    • IBM
  • Select The Language - Opción para seleccioner el idióma de para realizar Text To Speech
    • es - Español
    • en - Ingles
    • pt - Portugües

Input schema

  • ucid (Universal Call Identifier) (mandatory) - Identificador de llamada de la llamada donde se reproduce el anuncio. Asigne el ucid desde el appid del evento Call o desde el esquema de salida de la tarea Make Call.
  • mediauri - Texto de entrada para crear el audio que se reproducirá en la llamda o URL para obtener el archivo de audio
  • handle - El número / dirección para quien se reproducirá el anuncio.
  • participant - El destinatario del anuncio: la parte llamada o la parte que llama. Entradas válidas: callid, llamada, llamada.
  • interruptibility (optional) - Determina si el anuncio de audio se interrumpe cuando se presiona un dígito. Verdadero (predeterminado), interrumpe el anuncio. Falso no lo hace.
  • duration - La duración de una solicitud de llamada en segundos.
  • iterateCount - El número de veces que se reproduce el anuncio.
  • announcementForever (Optional) - Configura los medios para que se reproduzcan para siempre sin ser interrumpidos por el vencimiento de un temporizador. Los parámetros de duración y recuento de iteraciones no se utilizan cuando este indicador se establece en verdadero. El valor predeterminado es falso.
  • locale - Lenguaje de solicitud admitido por Avaya Aura® Media Server (MS).
  • useLocale - Se aceptan valores verdaderos y falsos. Si es verdadero, la asignación / propiedad de la configuración regional se utiliza para reproducir medios. Si no se proporciona, se utiliza el atributo de configuración regional predeterminado de System Manager.
  • language - Opción para seleccioner el idióma de para realizar Text To Speech
  • cloudProvider- Opción para seleccioner el tipo de proveedor para realizar Text To Speech

Output schema

  • ucid - Identificador de llamada de la llamada.
  • status:
    • COMPLETE - LLamada completada exitosamente
    • FAILED - La llamada falló
    • INTERRUPTED - llamada interrumpida
    • STOPPED - llamada detenida

Casos de uso

La conversión texto-voz es la generación por medios automáticos de una voz artificial que genera el sonido producido por una persona al leer un texto cualquiera en voz alta. o una voz artificial. Es decir, son sistemas que permiten la conversión de textos en voz sintética. Los conversores de texto-voz son conocidos también con las siglas CTV o por las siglas en inglés TTS (Text To Speech).
