Traductor

Descripción de la tarea

Recibe texto y devuelve la traducción del texto de acuerdo al modelo que se especifica en la propiedad modelId

Propiedades de la tarea

  • Texto - Texto que se desea traducir
  • Model Id - Indicar el lenguaje de origen y el lenguaje destino, tomar de referencia el siguiente enlace Por ejemplo: "es-en" (Español - Ingles), "en-es" (Ingles - Español), "pt-en" (Portugües - Ingles), "en-pt" (Ingles - Portugües) "https://console.bluemix.net/docs/services/language-translator/translation-models.html#translation-models"
  • Api Key - Api Key proporcionada por IBM watson al crear el servicio desde el portal IBM bluemix

Input schema

  • Texto - Texto que se desea traducir

Output schema

  • traduccion - Despliega la traducción del texto especificado

Prerequisitos

Api Key proporcionada por IBM bluemix

Related Boundary Events

No aplica

Casos de uso

Traducir texto de un idioma a otro usando la IA de Watson Language Translator
